About PickTone

PickTone is a free online practice tool for musicians. It combines a chromatic tuner and a metronome in a single, lightweight web app — no downloads, no sign-ups, no ads (yet).

Features

  • Chromatic tuner with real-time pitch detection via your microphone
  • Adjustable A4 reference pitch (420–460 Hz)
  • Metronome with tap tempo and customizable time signatures
  • Mute individual beats for accent practice
  • Works with guitar, bass, and other instruments via audio interface
  • Bilingual interface (English / Japanese)
  • Settings saved locally — pick up where you left off

Technology

PickTone runs entirely in your browser using the Web Audio API. Pitch detection uses an autocorrelation algorithm. No audio data is sent to any server — everything is processed locally on your device.
